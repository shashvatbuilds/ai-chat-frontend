# Eternal Chat Frontend

Modern AI chat application frontend built using React, Tailwind CSS, and Fetch API.

---

# Features

- User Registration
- User Login
- JWT Authentication
- Protected Chat Routes
- AI Chat Interface
- Dynamic Conversation Sidebar
- Auto Conversation Titles
- Delete Conversations
- Real-time AI Responses
- Auto Scroll Chat
- Responsive Dark UI

---

# Tech Stack

- React
- Tailwind CSS
- React Router DOM
- Fetch API
- Vite
